# learn-rhf

React Hook Form を体系的に学習するための Next.js プロジェクト。

## 技術スタック

- Next.js 16 / React 19 / TypeScript
- Tailwind CSS v4
- React Hook Form + @hookform/resolvers + Zod

## 始め方

```bash
pnpm dev
```

`http://localhost:3000` を開くとフェーズ一覧が表示される。

## 学習ロードマップ

詳細は [LEARNING.md](./LEARNING.md) を参照。

| フェーズ | テーマ |
|----------|--------|
| 1 | 基礎固め（useForm / register / formState） |
| 2 | 実践力強化（Controller / useFieldArray） |
| 3 | 深堀り（非制御の仕組み / Zod連携） |
| 4 | 応用パターン（パフォーマンス / マルチステップ） |
